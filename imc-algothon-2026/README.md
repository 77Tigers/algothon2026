
# Install
```shell
pip install -r requirements.txt
```

# Run

1. Start the data fetching bot. (You will need to set AERODATABOX_KEY in env for airport data fetching)

```shell
python fetchbot.py
```

2. Start the trading bot with MarketMaking. (Detail of the market making algorith can be found in [Algothon_2026_IMC_Challenge.pdf](Algothon_2026_IMC_Challenge.pdf))

```shell
python mm_bot.py
```
